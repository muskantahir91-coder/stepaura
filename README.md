# Step Aura — Static Website

A premium 7-page e-commerce site for a shoe brand, built with **plain HTML, CSS & JavaScript**. No build tools required.

## How to use

1. Extract the ZIP file.
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox, Safari).
3. Browse the site — all pages and features work locally.

## Pages

- `index.html` — Home (hero, benefits, promos, featured products, categories, testimonials)
- `shop.html` — Shop all products (filter by category & price, sort)
- `categories.html` — Men, Women, Kids overview
- `about.html` — About us
- `contact.html` — Contact form
- `auth.html` — Sign in / Sign up
- `cart.html` — Cart & Checkout

## Features

- ✅ Responsive (mobile + desktop)
- ✅ Sticky navbar with cart counter
- ✅ Smooth animations & hover effects
- ✅ Cart with localStorage persistence
- ✅ Sign-up / Sign-in flow (must sign in to place an order)
- ✅ Discount badges, product zoom on hover
- ✅ Newsletter & contact forms (demo)
